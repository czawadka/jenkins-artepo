Name of promotion which contains artepo to be used as source artepo. For example if there is sequence of promotions ACC -> PRD then for promotion PRD we can set that source promotion is ACC. During PRD promotion it will cause to copy artifacts from ACC artepo to PRD artepo.
If name is not set then main level artepo will be used or else (if current artepo is a main level artepo) workspace will be used.
See Promoted Builds Plugin for more on concept of build promotions.
